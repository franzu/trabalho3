Sobre
	Este repositorio visa a confecção do trabalho de Linguagens de Programação e Aplicações (SCC-0300). Nele se encontra um arquivo escrito em linguagem C cujo conteudo trata de matrizes esparsas. 
	
Ambiente de Desenvolvimento
	Os codigos foram escritos no compilador (IDE) Dev-C++ em Windows 10 x64. A compilação foi feita pelo mingw32-gcc.exe, gcc 4.7.1 Windows/unicode - 32bit.

Conteúdo
	O programa oferece um menu em que se tem a opções de: criar a matriz; deleta-la; inserir elementos; consultar elementos e somar elementos duma linha ou duma coluna.
	Encontra-se no repositorio uma pasta com as bibliotecas ultizidadas (stdio.h e stdlib.h) e um arquivo .txt com entradas exemplo para o programa.
